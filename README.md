# 基于生成式 AI 的交通/工業數據可解釋化分析系統 (GenAI-Driven XAI for Traffic/Industrial Data Analysis)

## 摘要：
在工業 4.0 與智慧交通系統中，數據分析往往面臨「模型準確但缺乏解釋性」以及「非技術人員難以解讀原始數據」的挑戰。針對本實驗室關注的大數據分析流程與可解釋化 AI（XAI）需求，本專題提出並實作了一套智慧數據分析助理。

系統透過 Streamlit 建構，允許使用者上傳感測器時間序列數據（如交通流量或機台震動數值）。後端首先利用 Python 進行自動化特徵提取與異常偵測（Data Cleaning & Feature Engineering），隨後將統計特徵轉化為自然語言提示（Prompt），輸入至大型語言模型（LLM）。LLM 扮演資深資料工程師的角色，針對數據波動提供邏輯推論與決策建議。本實作驗證了 AIGC 技術如何輔助傳統數值分析，將複雜的數據趨勢轉化為直觀的決策報告，為未來整合客製化神經網路預測模型奠定了人機協作的基礎。
